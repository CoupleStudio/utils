# utils
#开发小工具合集--nodejs     

## copyDirAndFiles.js       

>功能：复制指定文件夹下所需要的文件。文件夹架构不变。支持文件格式过滤.    

>用法：命令行下输入参数：1.脚本名 2.复制文件夹路径 3.需要复制的文件格式（为空时复制全部文件夹）

## listDirAndFiles.js       

>功能：列出指定文件夹架构及所含文件的路径.    

>用法：命令行下输入参数：1.脚本名 2.指定文件夹路径     

## picDetail.js       

>功能：指出png、jpg文件的尺寸.    

>用法：命令行下输入参数：1.脚本名 2.指定文件夹路径 3.文件格式
